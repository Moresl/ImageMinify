# 常见问题解答 (FAQ)

## 一般问题

### 这个工具是免费的吗？

是的，ImageMinify 是完全免费和开源的，根据 MIT 许可证发布。

### 支持哪些操作系统？

ImageMinify 支持 Windows 10 及以上版本（基于 WPF 框架）。

### 需要安装 .NET 运行时吗？

如果您下载的是自包含（Self-Contained）版本，则不需要额外安装。如果下载的是依赖框架版本，需要安装 [.NET 10 运行时](https://dotnet.microsoft.com/download/dotnet/10.0)。

## 功能问题

### 支持哪些图片格式？

- **输入格式**：JPEG (.jpg, .jpeg)、PNG (.png)、BMP (.bmp)、WebP (.webp)
- **输出格式**：JPEG、PNG、WebP

### 压缩是否会降低图片质量？

取决于您选择的格式和设置：

- **JPEG**：有损压缩，可通过质量滑块控制压缩程度
- **PNG**：使用 imagequant 量化时为有损压缩（视觉差异通常不明显），纯 ImageSharp 为无损压缩
- **WebP**：支持无损模式

### 压缩后的文件保存在哪里？

压缩后的文件保存在原文件所在的目录中：

- 未启用重命名时，文件名添加 `_compressed` 后缀
- 启用重命名时，按指定格式（前缀 + 分隔符 + 序号）命名

### 有文件大小限制吗？

理论上没有严格限制，但处理超大图片（如 50MB 以上）可能导致内存使用增加和处理时间延长。

### 什么是 imagequant 和 MozJPEG？

- **imagequant**：高质量 PNG 量化库，能显著减小 PNG 文件体积同时保持视觉质量
- **MozJPEG**：Mozilla 开发的 JPEG 编码器，可在不损失质量的情况下进一步减小 JPEG 文件体积

这些是可选的原生库，未安装时程序使用 ImageSharp 内置压缩，仍可正常工作。

## 技术问题

### 为什么某些 PNG 文件压缩效果不明显？

PNG 是已经压缩的格式。对于已优化过的 PNG 或包含大量不同颜色的图片，进一步压缩空间有限。使用 imagequant 量化可以获得更好的压缩效果。

### 我的杀毒软件报警了怎么办？

如果您从官方 Release 下载，程序是安全的。部分杀毒软件可能对自包含 .NET 应用产生误报。您可以查看源代码进行验证，或将程序添加到杀毒软件的白名单中。

### 应用程序崩溃了怎么办？

请尝试以下步骤：

1. 重新启动应用程序
2. 尝试处理较少的文件
3. 检查是否有足够的磁盘空间
4. 如果问题持续，请在 [GitHub Issues](https://github.com/Moresl/ImageMinify/issues) 报告，并提供详细的错误信息和重现步骤

### 设置保存在哪里？

设置保存在 Windows 注册表中，下次启动时自动恢复。

请查看[开发者文档](developer_guide.md)了解构建方法。

### 之前的 Python 版本还能用吗？

Python 版本（v1.x）的代码仍保留在 Git 历史中。但建议使用最新的 C# 版本（v2.0+），性能和功能都有显著提升。

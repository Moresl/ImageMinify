# 用户手册

本文档提供了 ImageMinify 图片压缩工具的详细使用说明。

## 界面概览

ImageMinify 采用 Fluent Design 风格界面，主要包含以下区域：

1. **文件选择区域**：选择要压缩的目录或文件
2. **压缩设置区域**：设置输出格式和压缩参数
3. **重命名设置区域**：配置文件重命名选项
4. **压缩按钮**：开始压缩过程
5. **结果表格**：显示压缩结果
6. **统计摘要**：显示总体压缩统计信息
7. **引擎能力标签**：显示当前可用的压缩引擎

## 基本操作

### 选择文件或目录

您可以通过两种方式选择要压缩的图片：

- **选择目录**：点击"选择目录"按钮，选择包含多个图片的文件夹
- **选择文件**：点击"选择文件"按钮，选择一个或多个特定的图片文件

### 选择输出格式

- **保持原格式**：保持原始格式但进行优化压缩
- **JPEG**：转换为 JPEG 格式（可通过滑块调整质量，范围 1-100）
- **PNG**：转换为优化的 PNG 格式
- **WebP**：转换为 WebP 格式

### 调整压缩质量

使用质量滑块可以调整压缩强度：
- 数值越高，图片质量越好，文件越大
- 数值越低，压缩率越高，但质量会下降
- 建议 JPEG 使用 75-85 的质量值

### 文件重命名

如果需要对压缩后的文件进行重命名：

1. 勾选"启用重命名"
2. 设置前缀（如 `photo`）
3. 选择分隔符（如 `_`）
4. 文件将按 `前缀_序号.扩展名` 的格式命名

### 开始压缩

点击"开始压缩"按钮，程序将：
1. 按选定格式和质量逐一压缩图片
2. 在结果表格中实时显示每张图片的压缩结果
3. 完成后显示总体统计信息（总数、成功数、总压缩比等）

## 压缩引擎

ImageMinify 根据可用的原生库自动选择最佳压缩引擎：

| 格式 | 基础引擎 | 增强引擎（可选） |
|------|----------|------------------|
| PNG | ImageSharp | imagequant（量化 + 抖动）|
| JPEG | ImageSharp | MozJPEG（无损优化）|
| WebP | ImageSharp | - |

界面底部的引擎能力标签会显示当前可用的压缩引擎状态。

## EXIF 信息

ImageMinify 可以读取图片的 EXIF 元数据信息，包括：
- 拍摄设备
- 拍摄时间
- 分辨率
- GPS 位置等

## 设置保存

您的偏好设置（输出格式、质量值、重命名选项等）会自动保存，下次启动时自动恢复。

## 提示

- 压缩后的文件默认保存在原文件所在目录
- 建议在压缩前备份重要的原始图片
- PNG 量化压缩属于有损压缩，但视觉差异通常不明显
- 处理大量文件时请耐心等待
